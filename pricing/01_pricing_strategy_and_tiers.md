### Pricing Strategy & Tiers

This doc defines the pricing model for Context Ark.

---

### 1. Free Tier (Try it out)

* **Price:** $0 / month.
* **Includes:**
  * 1 Active Project.
  * "Economy" AI Model Profile (fast/cheap models).
  * Max 10 docs generated per project.
  * Manual export only (no API access).
* **Goal:** Let users experience the "magic moment" of generating a kernel + first few docs.

---

### 2. Starter Tier (Solo Builder)

* **Price:** $19 / month.
* **Includes:**
  * 3 Active Projects.
  * "Standard" AI Model Profile (high-quality reasoning).
  * Full 50-doc pack generation.
  * Basic Email Support.
* **Goal:** Perfect for indie hackers and freelancers.

---

### 3. Pro Tier (Power User)

* **Price:** $49 / month.
* **Includes:**
  * Unlimited Projects.
  * Priority access to newest models (e.g., GPT-4o, Claude 3.5 Sonnet).
  * Team features (shared projects - Future).
  * API Access for automation.
* **Goal:** For heavy users and small agencies.

---

### 4. BYOK (Bring Your Own Key)

* **Price:** Free (Community Edition) or small platform fee.
* **Mechanism:**
  * User enters their own OpenAI/Anthropic API Key.
  * They pay for usage directly to the provider.
  * We charge $0 markup on tokens.
* **Goal:** For privacy-conscious users or heavy users who want direct cost control.
